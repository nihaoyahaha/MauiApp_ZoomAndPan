### ZoomPanContainer 是一个基于 .NET MAUI（跨平台 UI 框架）自定义的容器控件，用于实现对子内容（Content）的 缩放（Zoom） 和 平移（Pan / 拖拽） 功能。它结合了 PinchGestureRecognizer（捏合手势）和 PanGestureRecognizer（拖动手势），让用户可以通过双指缩放、单指拖动来查看内容，常用于图片查看器、地图、图表等需要交互式浏览大尺寸内容的场景。

🧩 核心功能
## 缩放（Zoom）：
### 通过双指捏合手势放大/缩小内容。
### 缩放范围限制在 [0.25, 8] 倍之间（防止过度缩小或放大）。
### 缩放中心基于用户手指捏合的起始点（ScaleOrigin），实现“以焦点为中心缩放”。
## 平移（Pan / Drag）：
### 通过单指拖动移动已缩放的内容。
### 仅在内容超出容器可视区域时才允许平移（即“越界”时才可拖动）。
## 边界限制（Clamping）：
### 自动计算内容在缩放后的有效平移范围，防止内容被拖出可视区域。
### 区分内容是否“越界”（宽/高大于容器），分别处理居中与可拖动逻辑。
